# My Coding challenges

This repository contains solutions for the weekly coding challenges from [Coding Challenges](https://codingchallenges.fyi/), curated by John Cricket. These challenges simulate real-world problems, offering opportunities to enhance software development skills, learn new technologies, and refine techniques.

My solutions are implemented in Java, reflecting my experience with the language since version 8.0 in 2012.

For more solutions from the community, check out [Shared Solutions](https://github.com/CodingChallengesFYI/SharedSolutions).


## Developer Guide

Prerequisite

  - git
  - jdk 21    
     *Defined JAVA_HOME enrivonment variable in your system*
  - maven
